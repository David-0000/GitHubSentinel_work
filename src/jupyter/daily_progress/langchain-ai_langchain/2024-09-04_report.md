# 项目简报：langchain-ai/langchain (2024-09-04)

## 新增功能
- **OllamaFunctions**：添加了 `bind_tools` 和 `with_structured_output` 函数，增强了结构化输出的功能。（#20881）

## 主要改进
- **Python 支持**：正式停止对 Python 3.8 的支持，以适应新的开发需求。（#25996）
- **Pydantic 升级**：将 Pydantic 版本升级至 v2，以提高数据验证和处理的效率。（#25986）
- **环境变量支持**：增加了对多个环境变量键的支持，用于从环境变量中提取秘密。（#25971）

## 修复问题
- **MongoDB 文档获取**：限制了从 MongoDB 数据库获取的最近文档数量，优化了数据处理效率。（#25435）
- **执行错误**：修复了在 `_is_field_useful` 函数中可能出现的执行错误。（#25095）
- **Azure 嵌入 API**：恢复了 Azure 嵌入 API 的 `api_version` 别名，确保了 API 的兼容性。（#26003）
- **属性错误**：修复了 `langchain-core` 中的属性错误问题。（#25953）

## 其他更新
- **测试和基础设施**：更新了测试框架，移除了 Pydantic v1 的测试，并调整了 CI 配置以适应新的发布候选版本。（#26006, #26005, #25992, #25988）
- **版本发布**：发布了多个组件的新版本，包括 `core`, `community`, `milvus`, `text-splitters`, `mistralai`, 和 `langchain`。（#25987, #25982, #25981, #25979, #25978, #25977）
- **模型参数修复**：修正了 `HuggingFacePipeline` 中的 `model_id` 参数问题。（#25962）
- **环境变量配置**：允许通过环境变量设置 `ChatMistralAI` 的基本 URL。（#25956）

以上是今日项目的最新进展，涵盖了功能新增、主要改进和问题修复等多个方面，确保了项目的稳定性和功能的持续优化。
